🚀 PART 1: Razorpay Setup
1. Enable Subscriptions
Go to Razorpay Dashboard
Settings → Subscriptions → Enable
2. Create Plan (₹249/month)

Example API:

POST /v1/plans
{
  "period": "monthly",
  "interval": 1,
  "item": {
    "name": "Premium Plan",
    "amount": 24900,
    "currency": "INR"
  }
}

👉 Save plan_id

3. Create Subscription (with delay)
POST /v1/subscriptions
{
  "plan_id": "plan_xxx",
  "customer_notify": 1,
  "total_count": 12,
  "start_at": 1710000000
}

👉 start_at = current time + 1 day (trial)

🔐 PART 3: Backend (VERY IMPORTANT)

Using Node.js:

app.post("/create-subscription", async (req, res) => {
  const startAt = Math.floor(Date.now() / 1000) + (24 * 60 * 60);

  const response = await axios.post(
    "https://api.razorpay.com/v1/subscriptions",
    {
      plan_id: "plan_xxx",
      total_count: 12,
      start_at: startAt
    },
    {
      auth: {
        username: "KEY_ID",
        password: "SECRET"
      }
    }
  );

  res.json(response.data);
});
