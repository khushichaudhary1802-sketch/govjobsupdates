# Government Jobs Mobile App (React Native) – Full Guide

## 1. Overview

This app helps users find government jobs based on:

* State
* District
* Job Type (multiple selections)

Core features:

* Personalized job feed
* Bookmark jobs
* Apply via official website (opens inside app)
* ₹1 trial (1 day)
* ₹249/month recurring subscription

Built using:

* React Native (Expo recommended for Replit)
* Firebase (Auth + Firestore)
* Web scraping/API for job data
* Razorpay / Stripe for payments

---

## 2. User Flow

### Step 1: App Open

Ask user:

* Job listings based on preferences

## 7. Home Screen Example

```js
import React, { useEffect, useState } from 'react';
import { View, Text, FlatList } from 'react-native';
import { fetchJobs } from '../services/api';

export default function HomeScreen() {
  const [jobs, setJobs] = useState([]);

  useEffect(() => {
    fetchJobs().then(setJobs);
  }, []);

  return (
    <View>
      <FlatList
        data={jobs}
        renderItem={({ item }) => (
          <Text>{item.title}</Text>
        )}
      />
    </View>
  );
}
```

## 9. Bookmark Feature

Store bookmarks in Firebase:

```js
import { getFirestore, collection, addDoc } from 'firebase/firestore';

const db = getFirestore();

export const saveBookmark = async (job) => {
  await addDoc(collection(db, 'bookmarks'), job);
};
```
